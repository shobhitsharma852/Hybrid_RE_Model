# Deployment

## Overview
This application is designed to be safely deployed as a Streamlit web app for internal or external use.

The deployment setup ensures:
- No sensitive data is stored permanently
- Uploaded files are processed temporarily
- Demo data is safe for public access

---

## Environment Requirements

- Python 3.10 or higher
- Virtual environment recommended
- Supported OS: Windows, Linux, macOS

---

## Installation

1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies:

```bash
pip install -r requirements.txt
